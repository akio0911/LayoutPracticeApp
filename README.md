# LayoutPracticeApp
- SwiftUIを用いて、複数端末対応する方法を練習するアプリ


## iPhone 14におけるDemo

- iPad

  - iPad Air 10.9 inch(5gen) 1640 * 2360 (820/1136)
- iPhone
  - iPhone 14 Pro Max 1290 * 2796 (430/839)
  - iPhone 14 1170 * 2532 (390/763)
